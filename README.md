## Introduction

The Efinity Indexer is a project that uses Subsquid to extract, transform and serve blockchain data.

## Prerequisites

- Node 16.x
- Docker
- NPM

## Quick-start (using Docker)

1. Clone the repository
2. Run `docker compose up -d`
